##Travel Booking System (Backend)

A simple Travel Booking System backend application built using Spring Boot.
This project demonstrates a clean 3-tier architecture with REST APIs for managing travel bookings.
##Features
 Search Trips (Source → Destination)
 View Available Trips
 Book a Trip
 Cancel Booking
 Refund Processing
 Itinerary Management (Confirmed / Cancelled)
Architecture

##This project follows a 3-tier architecture:

Controller Layer → Handles API requests
Service Layer → Business logic
Repository Layer → Database interaction
Database → H2 (In-Memory)
##Tech Stack
Java
Spring Boot
Spring Data JPA
H2 Database
Maven
